# 2025-11-27 禁用 Tesla 弃用警告

## 任务目标

解决编译时出现的 Tesla 弃用警告，该警告来自传递依赖 `resend` 库。

## 计划阶段

### 问题分析

- **警告信息**：`use Tesla.Builder` 和 `use Tesla` 已被软弃用，将在未来主版本中移除
- **根本原因**：`resend` 库依赖 Tesla，并在内部使用了已弃用的 Tesla API
- **影响范围**：仅编译警告，不影响运行时功能
- **解决方案**：按照警告提示，在 `config/config.exs` 中添加配置禁用警告

### 技术方案

在 `config/config.exs` 中添加：

```elixir
config :tesla, disable_deprecated_builder_warning: true
```

### 风险评估

- **低风险**：仅禁用警告，不改变任何功能
- **合理性**：项目不直接使用 Tesla，警告来自第三方依赖，禁用警告是合理做法
